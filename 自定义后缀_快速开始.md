# 自定义订阅链接后缀 - 快速开始

## 🚀 立即使用

### 步骤 1：数据库迁移
```bash
python migrate_add_custom_slug.py
```

### 步骤 2：重启应用
```bash
python app.py
```

### 步骤 3：设置自定义后缀

1. 登录管理面板
2. 进入"用户管理"
3. 点击某个用户的"✏️ 编辑"
4. 在"自定义链接后缀"字段输入，例如：`my-link`
5. 保存

## ✨ 功能亮点

- ✅ 两种链接都能使用（向后兼容）
- ✅ 自定义后缀更易记
- ✅ 前端显示 🔗 标识自定义链接
- ✅ 格式自动验证（只允许字母、数字、下划线、中划线）
- ✅ 唯一性检查（不会重复）

## 📋 格式要求

**允许的字符：**
- 字母：a-z, A-Z
- 数字：0-9
- 下划线：_
- 中划线：-

**有效示例：**
- ✅ `my-custom-link`
- ✅ `family_sub`
- ✅ `user001`
- ✅ `vip-premium-2024`

**无效示例：**
- ❌ `我的链接` (不支持中文)
- ❌ `my link` (不支持空格)
- ❌ `user@001` (不支持特殊符号)

## ❓ 常见问题

**Q: 必须设置自定义后缀吗？**  
A: 不必须，留空将继续使用系统生成的长链接。

**Q: 设置后原链接会失效吗？**  
A: 不会，两个链接都可以正常使用。

**Q: 可以随时修改吗？**  
A: 可以，编辑用户时随时修改或清空。

**Q: 如何知道哪些是自定义链接？**  
A: 用户管理页面中，自定义链接前会显示 🔗 图标。
